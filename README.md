# 💡 Lighting Store 訂單系統 (Java Swing)

這是我第一次使用 **Java Swing / JFrame** 製作的圖形化訂單系統。  
主要功能是讓使用者輸入姓名、選擇會員、調整燈具數量，並即時顯示時間與列印訂單。

---

## 🧭 專案介紹

這個小專案模擬一間燈具店的訂購介面。  
使用者可以：

- 點選「+」「–」按鈕調整商品數量（吊燈、壁燈、立燈）  
- 勾選是否為會員  
- 點擊「確定」查看訂單結果  
- 使用「清除」或「離開」按鈕重設或結束程式  
- 螢幕上方會即時顯示目前時間  

整體設計以 **簡潔與操作直覺** 為目標。

---

## ✨ 功能重點

| 功能 | 說明 |
|------|------|
| 🕒 即時時間顯示 | 使用 `Timer` 每秒更新目前時間 |
| ➕➖ 商品加減 | 數量不會小於 0，保持整數 |
| 🧹 清除 | 一鍵清除所有輸入 |
| 🖨️ 列印 | 可直接列印訂單內容 |
| 🚪 離開 | 結束程式 |
